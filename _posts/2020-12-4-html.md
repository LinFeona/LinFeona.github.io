
# Html+css 笔记


``` css
text-align: center;//文字居中
----------边框
border: 5px solid red;//为块面增加边框 不定义方向默认四个方向
border-left: 10px solid red; //给框上边加线 
dotted: //定义一个点线边框
dashed: //定义一个虚线边框
solid: //定义实线边框

float:  left; //内容衔接在左边
margin: 10px; // 两个块之间的间距 
padding: 20px; // 内容上下左右都隔开边缘20 像素
font-family: Arial;// 字体
font-weight:bold;  // 加粗
color: yellow; //颜色
 ----------背景
background: url('url') no-repeat;
background-size: 100% 200%; // 背景大小
background-color: rgb(0,0,0,0.5);// 背景色

//圆角
	border-top-left-radius: 50px;

	 border-bottom-right-radius: 50px;
	 
	   text-shadow: 5px 5px 5px; 文字阴影
	 display: block; 行级 转块级

	块级 元素位置定位
	position: absolute; //在中间
	top: 10px;  顶端下10的位置
	right: 20px; 
```
